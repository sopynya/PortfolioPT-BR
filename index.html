DinoDatabase

DinoDatabase é um site informativo e responsivo voltado à divulgação científica sobre dinossauros. Desenvolvido com React.js, CSS, json e foco em experiência do usuário, o projeto apresenta uma vasta lista de dinossauros e suas informações, com conteúdo organizado, visual atrativo e URLs amigáveis.
O objetivo é tornar o conhecimento sobre paleontologia acessível e interessante para públicos diversos.

Hana-Gate

Hana-Gate é uma database de visual novels (jogos japoneses) que permite aos usuários criar conta, salvar títulos, favoritar obras, atribuir notas e compartilhar comentários.
A aplicação foi construída utilizando React.js, Supabase (para autenticação e banco de dados) e CSS, incorporando referências ao estilo japonês, ícones personalizados, seções bem organizadas e uma navegação fluida e agradável.

Eco Snap

Esse projeto foi feito usando React.js, Supabase para login e banco de dados, e CSS para o visual.
É um site de jogo onde o usuário usa um gacha para ganhar peixes, cada um com raridades diferentes. Esses peixes podem ser equipados e ajudam a aumentar a sorte do jogador, influenciando os próximos pulls e a progressão no jogo. A ideia foi criar algo simples de usar, com visual limpo e mecânicas fáceis de entender, mas que ainda deem vontade de continuar jogando.
