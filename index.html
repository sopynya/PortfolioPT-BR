Dino API

É uma API de dinossauros desenvolvida com Next.js, pensada como uma base de dados científica com informações organizadas sobre espécies, períodos geológicos e características principais. O projeto explora rotas de API, estrutura moderna do Next e organização de dados para facilitar consultas e uso educacional.


Live URL GitHub

DinoDatabase

DinoDatabase é um site informativo e responsivo voltado à divulgação científica sobre dinossauros. Desenvolvido com React.js, CSS, json e foco em experiência do usuário, o projeto apresenta uma vasta lista de dinossauros e suas informações, com conteúdo organizado, visual atrativo e URLs amigáveis.
O objetivo é tornar o conhecimento sobre paleontologia acessível e interessante para públicos diversos.


Live URL GitHub

Frontpage

Front Page é um agregador de conteúdo personalizável que reúne feeds RSS e Atom em um único dashboard de leitura bem estruturado e visualmente organizado. A aplicação permite centralizar diferentes fontes de informação, acompanhar novos artigos e criar uma experiência de leitura personalizada.
O projero foi criado com NextJs, Zustand, Tanstack Query, RSS parser e um JSON para alimentar o guest feed.


Live URL GitHub

Fishis

Fishis é um jogo de pesca desenvolvido com Next.js onde o usuário coleta peixes, vende suas capturas e evolui seu equipamento, podendo equipar varas e hooks para melhorar o desempenho. O projeto utiliza TanStack Query para gerenciamento de dados assíncronos, Zustand para estado global, PostgreSQL como banco de dados (hospedado no Neon) e autenticação via JWT. A aplicação foi construída com foco em organização, escalabilidade e arquitetura moderna full-stack.
