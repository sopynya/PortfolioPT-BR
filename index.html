DinoDatabase

DinoDatabase é um site informativo e responsivo voltado à divulgação científica sobre dinossauros. Desenvolvido com HTML, CSS e foco em experiência do usuário, o projeto apresenta páginas individuais para diferentes espécies, com conteúdo organizado, visual atrativo e URLs amigáveis.
Utiliza boas práticas de SEO, design responsivo e estilização moderna com fontes customizadas e gradientes suaves. O objetivo é tornar o conhecimento sobre paleontologia acessível e interessante para públicos diversos.

Pet Shop Landing page

Este projeto é uma landing page fictícia desenvolvida para um pet shop, com foco em design responsivo, visual limpo e comunicação clara com o usuário. A página foi criada com HTML, CSS e JavaScript puro, incorporando elementos modernos, ícones personalizados, seções bem definidas e navegação fluida.
O objetivo do site é apresentar os principais serviços do pet shop, incentivar o agendamento de visitas e reforçar o carinho e cuidado com os pets. O layout é totalmente adaptável para dispositivos móveis, com atenção à acessibilidade e usabilidade.

Great Recipes

Este projeto utiliza React e Vite, com foco em uma experiência interativa e intuitiva. O projeto permite aos usuários explorar e filtrar receitas de diferentes regiões do mundo, com opções de pesquisa por tipo (vegano, não vegano, bebidas) e sabor (doce, salgado).
React Router para navegação dinâmica, este projeto oferece uma interface responsiva e fluida, com imagens ilustrativas para cada receita. É uma ótima demonstração de como integrar funcionalidades de pesquisa e filtragem em uma SPA (Single Page Application).
